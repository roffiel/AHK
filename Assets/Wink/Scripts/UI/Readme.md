## Contenido

* **SubtitleDisplayer.cs** - Esta clase se encarga de desplegar subtitulos en el GUI.
* **Tools.cs** - Algunos métodos útiles para determinar los atributos de la pantalla.
  * `public static float GetScreenDiagonal()` - Gets the screen diagonal in inches.
  * `public static float GetScreenRatio()` - Returns the screen ratio.
  * `public static bool IsTablet()` - Determina que el dispositivo es una table cuando su pantalla es mayor a 6".
